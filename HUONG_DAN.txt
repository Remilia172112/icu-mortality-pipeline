================================================================
  CÁC BƯỚC THỰC HIỆN
================================================================

  BƯỚC 1 — Cài Python 3.10
  ─────────────────────────
  Tải tại: https://www.python.org/downloads/release/python-31011/
  → Chọn "Windows installer (64-bit)"
  → Khi cài: TÍCH "Add Python to PATH" ở màn hình đầu tiên
  → Bấm "Install Now"

  Kiểm tra: mở Command Prompt → gõ: python --version
  Phải hiện: Python 3.10.x


  BƯỚC 2 — Chỉnh config.txt
  ──────────────────────────
  Mở file config.txt bằng Notepad, sửa 2 dòng:

    DATA_DIR=C:\mimic4          ← đường dẫn MIMIC-IV của bạn
    OUT_DIR=C:\mimic4_processed ← nơi lưu kết quả (~60GB trống)

  Lưu lại (Ctrl+S)


  BƯỚC 3 — Cài môi trường (1 lần duy nhất)
  ──────────────────────────────────────────
  Double-click: 1_setup.bat
  Đợi hiện "[XONG] Cai dat thanh cong!" (~5 phút)


  BƯỚC 4 — Chạy xử lý dữ liệu
  ─────────────────────────────
  Double-click: 2_run.bat

  Thời gian ước tính (máy 32GB RAM):
  ┌──────────────────────────────────────────┐
  │ Step 1 - Cohort             :  ~2 phút   │
  │ Step 2 - chartevents        :  ~1-2 giờ  │
  │ Step 3 - labevents          :  ~30 phút  │
  │ Step 4 - inputevents (vaso) :  ~5 phút  │
  │ Step 5 - output + procedure :  ~5 phút   │
  │ Step 6 - Merge & tensor     :  ~10 phút  │
  │                                          │
  │ TỔNG: khoảng 5-6 giờ                     │
  └──────────────────────────────────────────┘

  Trong lúc chờ: ĐỪNG TẮT MÁY, ĐỪNG ĐÓNG CỬA SỔ
  Máy tính vẫn dùng bình thường được


  BƯỚC 5 — Kiểm tra kết quả
  ──────────────────────────
  Double-click: 4_verify.bat

  Kết quả đúng:
    X_train: (N, 48, 69)   mortality ~15%
    NaN trong X_train: 0
    >>> Du lieu sach! San sang train LSTM / Transformer.


================================================================
  NẾU BỊ NGẮT GIỮA CHỪNG (mất điện, treo máy...)
================================================================

  Pipeline tự động lưu cache sau mỗi bước:
    chart_wide.parquet  → sau Step 2 (bước tốn thời gian nhất)
    lab_wide.parquet    → sau Step 3
    vaso_wide.parquet   → sau Step 4
    urine_crrt.parquet  → sau Step 5

  → Double-click 3_rerun_from_cache.bat
  → Tự động phát hiện bước nào đã xong, bỏ qua, chạy tiếp
  → Thường chỉ mất thêm 20-30 phút để hoàn tất


================================================================
  KẾT QUẢ ĐẦU RA
================================================================

  Trong OUT_DIR sau khi chạy xong:

  X_train.pt          ← (N_train, 48, 69) — dùng để train model
  y_train.pt          ← (N_train,) — label 0/1
  X_val.pt / y_val.pt ← validation set
  X_test.pt / y_test.pt ← test set

  scaler.pkl          ← StandardScaler — lưu để dùng khi inference
  feature_names.json  ← tên và phân nhóm 69 features
  cohort.parquet      ← thông tin gốc ICU stays + labels
  chart_wide.parquet  ← cache (giữ để chạy lại nhanh)
  lab_wide.parquet    ← cache
  vaso_wide.parquet   ← cache
  urine_crrt.parquet  ← cache
